# 官方资料索引与复核清单

> 本文件只记录外部工具的官方资料入口和复核日期，不记录新闻、教程或临时结论。

## 复核规则

1. 涉及工具功能判断时，优先看官方文档。
2. 超过 30 天未复核的工具能力，不作为强结论。
3. 新闻、博客、教程只作参考，不作为单一事实来源。
4. ChatGPT Project / GPTS 知识库不能长期保存易过期外部结论。
5. 如果官方资料与本仓库旧判断冲突，优先修订本仓库旧判断。
