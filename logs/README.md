# Logs

`logs/` 用于保存学习过程和阶段复盘。

## 用途

- 记录每天学习 Agent OS 的产出。
- 记录 Hermes / Codex / Claude Code 对照判断。
- 记录哪些流程值得沉淀为 skill、quick command 或 automation。

## 不作为长期知识库

`logs/` 默认不上传 ChatGPT Project / GPTS 知识库。

原因：

- 它包含过程性判断。
- 它可能包含未确认结论。
- 它容易过期。

## 推荐命名

```text
logs/YYYY-MM/day01_为什么还要学Hermes.md
logs/YYYY-MM/day02_AGENTS大纲.md
logs/YYYY-MM/day03_memory边界.md
logs/YYYY-MM/day07_复盘.md
```

## 复盘模板

```text
日期：

今天学的概念：

它在 SK 里的用途：

Codex / Claude Code / Hermes 的分工判断：

今天产出的文件：

下一步：
```
