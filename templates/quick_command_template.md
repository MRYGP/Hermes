# Quick Command Template

## Command 名称

填写一个短命令名，例如：

```text
/sk-status-brief
```

## 目的

这个命令解决什么高频机械动作？

```text
快速生成 SK 当前工作上下文摘要。
```

## 输入

- 固定文件路径
- 可选参数
- 输出语言或格式

## 执行动作

1. 读取指定文件。
2. 提取固定字段。
3. 按固定格式输出。

## 输出格式

```text
当前阶段：

当前重点：

最近完成：

待确认：

下一步：
```

## 不做什么

- 不做开放式重构。
- 不自动修改文件。
- 不推断缺失事实。
- 不读取未指定的大量文件。

## 适合升级为 skill 吗

如果这个命令开始需要复杂判断、跨文件对比或错误处理，就不要继续塞进 quick command，应升级为 skill。
