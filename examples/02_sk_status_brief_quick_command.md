# Quick Command：SK 状态简报

## Command 名称

```text
/sk-status-brief
```

## 目的

快速生成当前仓库学习状态摘要，作为 ChatGPT Project 或 Codex 开始任务前的上下文入口。

## 输入

默认读取：

- `PROJECT_STATE.md`
- `README.md`
- `docs/00_项目定位.md`

## 执行动作

1. 读取指定文件。
2. 提取当前阶段、当前定位、已完成、未决问题、下一步。
3. 按固定格式输出。

## 输出格式

```text
当前阶段：

当前定位：

已完成：

未决问题：

下一步：

需要避免：
```

## 不做什么

- 不修改文件。
- 不跨仓库搜索。
- 不推断缺失事实。
- 不替代状态漂移审计。

## 升级条件

如果这个命令开始需要跨 5 个以上文件做冲突判断，应升级为 skill。
